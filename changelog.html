What’s New in DriveVault

Stay up to date with the latest features, improvements, and fixes.

New Features & Improvements v1.2.3 January 2026
Quick Search Projects Scanning Payments Stability
Highlights
  • Added macOS top menu Quick Search. Click the eye icon to open a powerful search bar separate from the main app window. Find any file instantly with advanced filters and sorting, without opening DriveVault.
  • Introduced Project sorting. Hover over any project to reveal up and down arrows, allowing you to reorder projects in any way that suits your workflow.
  • Launched a new, simpler non-subscription pricing model based on user feedback. Pay only for the number of drives you need, with the flexibility to grow into higher tiers or add individual drives over time.
  • Implemented new and improved scanning logic using intelligent drive partitioning to support larger drives and prevent stalled scans.
  • Major overhaul of the Manage Library system, including a powerful new library reset option. We recommend beta users run a reset to benefit fully from the new scanning engine or resolve persistent issues.

Improvements & Fixes

  • Overall stability improvements and extensive bug fixes across the app.
  • Fixed an issue where the Compare feature could incorrectly report 0% similarity. Backup drives are now correctly recognised, with reports generated for drives showing 80% similarity or higher.
  • Introduced new button colours and subtle glows for improved visual clarity and UI feedback.
  • Improved re-scanning behaviour so previously scanned drives can be reconnected and updated incrementally, adding only new files instead of rescanning entire drives.

As always, we recommend keeping DriveVault up to date to benefit from the latest performance improvements and fixes.

Improvements v1.13 October 1, 2025
Large Drive Scanning UI Performance Backup
Highlights
  • Fixed large drive scans failing or stalling at 50% with new auto-partitioning logic.
  • Redesigned Drive Scan UI with “Full Scan + Previews” default.
  • Added lazy loading for large libraries to prevent stalling.
  • Fixed column sorting for file size.
  • Added loading screen for Library Stats.
  • Smarter Backup Mode with refreshed duplicate detection.

Deeper Notes

  • New background partitioning handles drives above 2TB (200k+ files). Scans take ~3–6 mins per TB.
  • UI redesign aligns Drive Scan with DriveVault’s design standards.
  • Lazy loading ensures smooth navigation across massive catalogs.
  • Library Stats now shows a visible loading screen while compiling data.
  • Backup Mode refreshes automatically after each scan to highlight files not backed up (red) and duplicates (green).
  • Help information updated to reflect new features.
  • Removed legacy “High Capacity Mode” — replaced with intelligent auto-partitioning.

We’d love your feedback and screenshots — they help us make DriveVault the best it can be!
