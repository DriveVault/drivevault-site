New Features v0.9.8 January 2026
Quick Search Projects Scanning Pricing

Highlights

  • Added macOS top menu Quick Search. Click the eye icon to open a powerful search bar separate from the main app window. Find any file instantly with advanced filters and sorting, without opening DriveVault.
  • Introduced Project sorting. Hover over any project to reveal up and down arrows, allowing you to reorder projects in any way that suits your workflow.
  • Launched a new, simpler non-subscription pricing model based on user feedback. Pay only for the number of drives you need, with flexibility to grow.
  • Implemented new and improved scanning logic using intelligent drive partitioning to support larger drives and prevent stalled scans.
  • Major overhaul of the Manage Library system, including a powerful new library reset option.

Improvements & Fixes

  • Overall stability improvements and extensive bug fixes across the app.
  • Fixed an issue where the Compare feature could incorrectly report 0% similarity.
  • Introduced new button colours and subtle glows for improved visual clarity.
  • Improved re-scanning behaviour - previously scanned drives now update incrementally.
Improvements v0.9.5 October 2025
Large Drive Scanning UI Performance

Highlights

  • Fixed large drive scans failing or stalling at 50% with new auto-partitioning logic.
  • Redesigned Drive Scan UI with "Full Scan + Previews" default.
  • Added lazy loading for large libraries to prevent stalling.
  • Fixed column sorting for file size.
  • Added loading screen for Library Stats.
  • Smarter Backup Mode with refreshed duplicate detection.

Technical Notes

  • New background partitioning handles drives above 2TB (200k+ files).
  • Scans take approximately 3-6 mins per TB.
  • Lazy loading ensures smooth navigation across massive catalogs.
  • Removed legacy "High Capacity Mode" - replaced with intelligent auto-partitioning.
