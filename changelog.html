What’s New in DriveVault

Stay up to date with the latest features, improvements, and fixes.

New Features v1 – Public Beta September 22, 2025
Offline Library Search UI
Highlights
  • Public Beta launch with offline drive catalogs and global search.
  • Finder-style Column, Icon, and List views with quick previews.
  • Drive Library with custom icons, names, and online/offline status.

Deeper Notes

  • EXIF/metadata search (camera, lens, date taken) and file-type filters.
  • Smart Collections: Recent Photos, Large Files, and Unlabeled.
  • Analytics dashboard: library totals, file-type breakdown, capacity charts.
  • Performance: background scanning for 500k+ files and intelligent thumbnail caching.
  • Security: local-only storage; optional password lock (Starter+ preview).

Improvements v0.8.2 – Pre-Beta September 12, 2025
Performance Thumbnails
Highlights
  • 2x faster initial scans on large photo/video drives.
  • Smoother column navigation and better breadcrumb history.

Deeper Notes

  • Reduced memory spikes on libraries with many small files.
  • Improved EXIF extraction for HEIC/HEIF variants.
  • Fixed rare freeze when switching between Icon and List views quickly.
Fixes v0.7.0 – Alpha August 28, 2025
Stability UX
Highlights
  • Crash fixes during very large directory walks.
  • Refined empty states and tooltips across the app.

Deeper Notes

  • Improved error handling for disconnected network volumes.
  • Added first pass of drive personalization (custom icon & name).
  • Keyboard nav: ←/→ to move columns, ⌘K to jump to search.
