What’s New in DriveVault

Stay up to date with the latest features, improvements, and fixes.

Improvements v1.13 October 1, 2025
Large Drive Scanning UI Performance Backup
Highlights
  • Fixed large drive scans failing or stalling at 50% with new auto-partitioning logic.
  • Redesigned Drive Scan UI with “Full Scan + Previews” default.
  • Added lazy loading for large libraries to prevent stalling.
  • Fixed column sorting for file size.
  • Added loading screen for Library Stats.
  • Smarter Backup Mode with refreshed duplicate detection.

Deeper Notes

  • New background partitioning handles drives above 2TB (200k+ files). Scans take ~3–6 mins per TB.
  • UI redesign aligns Drive Scan with DriveVault’s design standards.
  • Lazy loading ensures smooth navigation across massive catalogs.
  • Library Stats now shows a visible loading screen while compiling data.
  • Backup Mode refreshes automatically after each scan to highlight files not backed up (red) and duplicates (green).
  • Help information updated to reflect new features.
  • Removed legacy “High Capacity Mode” — replaced with intelligent auto-partitioning.

We’d love your feedback and screenshots — they help us make DriveVault the best it can be!

New Features v1 – Public Beta September 22, 2025
Offline Library Search UI
Highlights
  • Public Beta launch with offline drive catalogs and global search.
  • Finder-style Column, Icon, and List views with quick previews.
  • Drive Library with custom icons, names, and online/offline status.

Deeper Notes

  • EXIF/metadata search and file-type filters.
  • Smart Collections: Recent Photos, Large Files, and Unlabeled.
  • Analytics dashboard with file-type breakdown and charts.
  • Performance: background scanning for 500k+ files.
  • Security: local-only storage; optional password lock.

Performance Thumbnails
