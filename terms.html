2 Description of Service

DriveVault is a drive cataloging application for macOS that allows you to:

  • Create offline catalogs of external drives through read-only scanning
  • Generate visual previews and extract metadata from your files
  • Search across all cataloged drives - even when disconnected
  • Organize drives into projects with custom icons, tags, notes, and ratings
  • Compare drives and generate library reports
  • Export and import drive catalogs for sharing or backup
  • Optionally create folder structures at user-selected locations

DriveVault creates catalogs for offline reference only. It does not copy, sync, or back up your files. You are solely responsible for maintaining proper backups of your data.

3 How DriveVault Works

DriveVault operates using read-only scanning within the macOS sandbox. The app reads your files to generate previews and extract metadata, but it never modifies or deletes your original files.

Optional write operations are strictly limited to user-authorized actions, such as creating folder structures at locations you explicitly choose. These operations never overwrite existing files.
